# My_Application_11
mobilki

Описание: Мобильное приложение для создания и прохождения тестов, позволяющее управлять тестами и вопросами, делиться ссылкой на прохождение и отслеживать результаты с аналитикой и статистикой.

Инструкция по установке:
1. Скопировать файл  
   - Передать apk на телефон.  
   - Запомнить папку, куда он сохранён (например, Download).

2. Разрешить установку из неизвестных источников  
   - Открыть «Настройки» → «Безопасность» или «Приложения».  
   - Найти пункт «Установка неизвестных приложений» / «Неизвестные источники».  
   - Выдать разрешение для того приложения, через которое будешь открывать apk (например, «Файлы», «Chrome», «Telegram»).

3. Установить apk  
   - Открыть файловый менеджер на телефоне.  
   - Найти apk-файл и нажать на него.  
   - В появившемся окне выбрать «Установить» и дождаться завершения.

4. Запустить приложение  
   - После установки можно нажать «Открыть» сразу.  
   - Либо найти иконку приложения в списке программ и запустить оттуда.
